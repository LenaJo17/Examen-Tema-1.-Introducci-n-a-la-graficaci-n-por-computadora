# 🌸 Flor Vitral - Canvas API

## 📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una aplicación web que utiliza la API Canvas 2D para dibujar por programación una figura inspirada en una flor tipo vitral.

La aplicación muestra:

* A la izquierda: la imagen dibujada por programación.
* A la derecha: la imagen original de referencia.

El objetivo fue aplicar los fundamentos de graficación por computadora utilizando figuras básicas como:

* Rectángulos
* Elipses
* Líneas
* Curvas cuadráticas
* Círculos

La ilustración fue construida completamente mediante código, superando el mínimo requerido de 30 figuras básicas.

## 🎯 Funcionalidades

✔ Dibujo programado de una figura usando Canvas
